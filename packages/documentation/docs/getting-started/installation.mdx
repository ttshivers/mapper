### Core

Install the `core` module via `npm` or `yarn` along with at least one strategy:

```shell
npm i @ttshivers/automapper-core @ttshivers/automapper-classes reflect-metadata # for classes
npm i @ttshivers/automapper-core @ttshivers/automapper-pojos # for pojos
```

```shell
yarn add @ttshivers/automapper-core @ttshivers/automapper-classes reflect-metadata # for classes
yarn add @ttshivers/automapper-core @ttshivers/automapper-pojos # for pojos
```

#### `tsconfig`

```js
{
    "skipLibCheck": true,
    "experimentalDecorators": true, // for @ttshivers/automapper-classes
    "emitDecoratorMetadata": true // for @ttshivers/automapper-classes
}
```

### ORMs

If you are using an ORM, AutoMapper TypeScript provides support for [Sequelize](https://sequelize.org/) and [MikroORM](https://mikro-orm.io/). Both work with `@ttshivers/automapper-classes`

```shell
npm i @ttshivers/automapper-core @ttshivers/automapper-classes reflect-metadata @ttshivers/automapper-mikro # for mikro-orm
npm i @ttshivers/automapper-core @ttshivers/automapper-classes reflect-metadata @ttshivers/automapper-sequelize # for sequelize
```

```shell
yarn add @ttshivers/automapper-core @ttshivers/automapper-classes reflect-metadata @ttshivers/automapper-mikro # for mikro-orm
yarn add @ttshivers/automapper-core @ttshivers/automapper-classes reflect-metadata @ttshivers/automapper-sequelize # for sequelize
```

### NestJS

AutoMapper TypeScript provides official integration with [NestJS](https://nestjs.com)

```shell
npm i @ttshivers/automapper-core @ttshivers/automapper-nestjs @ttshivers/automapper-classes reflect-metadata # for classes
npm i @ttshivers/automapper-core @ttshivers/automapper-nestjs @ttshivers/automapper-classes reflect-metadata @ttshivers/automapper-mikro # for classes + mikro-orm
npm i @ttshivers/automapper-core @ttshivers/automapper-nestjs @ttshivers/automapper-classes reflect-metadata @ttshivers/automapper-sequelize # for classes + sequelize
npm i @ttshivers/automapper-core @ttshivers/automapper-nestjs @ttshivers/automapper-pojos # for pojos
```

```shell
yarn add @ttshivers/automapper-core @ttshivers/automapper-nestjs @ttshivers/automapper-classes reflect-metadata # for classes
yarn add @ttshivers/automapper-core @ttshivers/automapper-nestjs @ttshivers/automapper-classes reflect-metadata @ttshivers/automapper-mikro # for classes + mikro-orm
yarn add @ttshivers/automapper-core @ttshivers/automapper-nestjs @ttshivers/automapper-classes reflect-metadata @ttshivers/automapper-sequelize # for classes + sequelize
yarn add @ttshivers/automapper-core @ttshivers/automapper-nestjs @ttshivers/automapper-pojos # for pojos
```
